# javascript-studies-es6


## Clone the repo & install
```
git clone https://github.com/suxxus-studies/javascript-studies-es6.git
cd ./javascript-studies-es6
npm install
```
## Usage
```
cd ./javascript-studies-es6
npm run dev

```
